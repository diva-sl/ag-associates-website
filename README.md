# AG & ASSOCIATES — Corporate Tax Consultancy Website

A premium, modern corporate website built for **AG & Associates**, a professional tax consultancy firm.  
Designed with a focus on trust, clarity, and high conversion — ideal for financial and consulting businesses.

---

## 🚀 Live Features

✅ Modern corporate UI  
✅ Fully responsive design  
✅ Smooth scrolling navigation  
✅ Premium animations with Framer Motion  
✅ Conversion-focused layout  
✅ Interactive service cards  
✅ Animated statistics  
✅ Contact form integration  
✅ Social media linking  
✅ Mobile-friendly navigation  
✅ Glassmorphism design  

---

## 🛠 Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS
- Framer Motion
- Lucide Icons

**UI Architecture**
- Component-based structure
- Reusable UI elements
- Utility-first styling
- Clean folder organization
